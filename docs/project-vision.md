# Buildora

Buildora is a modern visual drag-and-drop website builder.

## Target Users

- Freelancers
- Developers
- Small businesses

## Core Features

- Drag & Drop
- Responsive Preview
- Components
- History (Undo/Redo)
- Templates


## Future Improvements

- Additional Components
- Theme Marketplace
- More Export Formats
- Keyboard Shortcuts
- Auto Save
- Dark / Light Themes


## Future vision

- AI
- Collaboration
- Plugin System

## Non Goals

- WordPress replacement
- CMS
